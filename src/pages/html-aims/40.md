## Submit & Reset Buttons

**Description**

Submit buttons send form data to the server. Reset buttons clear all form fields. Use `<button type="submit">` for submission and `<button type="reset">` for clearing.

### Submit Button

```html
<button type="submit">Submit Form</button>
```

### Reset Button

```html
<button type="reset">Clear Form</button>
```

### Both Buttons Together

```html
<form action="/submit" method="POST">
  <input type="text" name="username" placeholder="Username" required>
  <input type="email" name="email" placeholder="Email" required>
  
  <button type="submit">Submit</button>
  <button type="reset">Clear</button>
</form>
```

### Button vs Input

Using `<button>` element is preferred:
```html
<!-- Recommended -->
<button type="submit">Submit</button>
<button type="reset">Reset</button>
```

But `<input>` also works:
```html
<!-- Still valid -->
<input type="submit" value="Submit">
<input type="reset" value="Reset">
```

### Real-World Examples

**Sign-up Form:**
```html
<form action="/signup" method="POST">
  <h2>Create Account</h2>
  
  <label for="username">Username:</label>
  <input type="text" id="username" name="username" required>
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  
  <label for="password">Password:</label>
  <input type="password" id="password" name="password" required>
  
  <div class="form-actions">
    <button type="submit">Create Account</button>
    <button type="reset">Clear</button>
  </div>
</form>
```

**Contact Form:**
```html
<form action="/contact" method="POST">
  <input type="text" name="name" placeholder="Your Name" required>
  <input type="email" name="email" placeholder="Your Email" required>
  <textarea name="message" placeholder="Message" required></textarea>
  
  <button type="submit">Send Message</button>
  <button type="reset">Clear</button>
</form>
```

### Attributes

- `type="submit"` - Submit form data
- `type="reset"` - Clear all fields
- `type="button"` - Generic button (use with JavaScript)
- `disabled` - Disable button
- `name` - Form field name (for submit buttons)

### Practice Tips

1. Use button element
2. Always specify type
3. Use clear labels
4. Consider disabling reset (users rarely use)
5. Style with CSS
