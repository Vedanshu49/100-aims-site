## Data Attributes

**Description**

Data attributes (`data-*`) store custom data in HTML elements without affecting styling or semantics. They're accessed via JavaScript and useful for storing element-specific information.

### Basic Data Attributes

```html
<div data-id="123" data-name="John">User</div>
<button data-action="delete" data-item-id="456">Delete</button>
<img data-gallery="photos" data-index="1" src="photo.jpg">
```

### Accessing Data Attributes in JavaScript

```html
<div id="user" data-id="789" data-role="admin">
  Admin User
</div>

<script>
  const user = document.getElementById('user');
  
  // Using dataset property
  console.log(user.dataset.id);      // "789"
  console.log(user.dataset.role);    // "admin"
  
  // Using getAttribute
  console.log(user.getAttribute('data-id'));  // "789"
</script>
```

### Real-World Examples

**Product Cards:**
```html
<div class="product" data-product-id="101" data-price="29.99" data-category="electronics">
  <h3>Laptop</h3>
  <p>$<span data-currency="USD">29.99</span></p>
  <button class="add-to-cart" data-action="add">Add to Cart</button>
</div>

<script>
  document.querySelector('.add-to-cart').addEventListener('click', (e) => {
    const productId = e.target.closest('.product').dataset.productId;
    const price = e.target.closest('.product').dataset.price;
    console.log(`Adding product ${productId} at $${price}`);
  });
</script>
```

**Gallery with JavaScript:**
```html
<div class="gallery">
  <img data-image-id="1" data-alt-text="Sunset" src="sunset.jpg" alt="Sunset">
  <img data-image-id="2" data-alt-text="Mountain" src="mountain.jpg" alt="Mountain">
  <img data-image-id="3" data-alt-text="Ocean" src="ocean.jpg" alt="Ocean">
</div>

<script>
  document.querySelectorAll('img').forEach(img => {
    img.addEventListener('click', () => {
      console.log(img.dataset.imageId, img.dataset.altText);
    });
  });
</script>
```

**Conditional Fields:**
```html
<form>
  <select id="type" data-field-type="select">
    <option>Option A</option>
    <option data-requires="extra-info">Option B (requires info)</option>
  </select>
  
  <input type="text" id="extra" data-field-name="extra-info" style="display:none">
</form>
```

### Naming Convention

- Use lowercase
- Separate words with hyphens (data-product-id)
- Access with camelCase in JavaScript (dataset.productId)
- Examples:
  - `data-user-id` → `dataset.userId`
  - `data-api-key` → `dataset.apiKey`
  - `data-is-active` → `dataset.isActive`

### Use Cases

1. Store element-specific IDs
2. Pass data to JavaScript handlers
3. Store API parameters
4. Configure behavior per element
5. Track analytics data

### Practice Tips

1. Use semantic HTML first
2. Data attributes complement HTML
3. Access via dataset property
4. Use for JavaScript interaction
5. Validate data server-side
