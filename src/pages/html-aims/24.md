## Nested Lists

**Description**

Nested lists are lists within lists. They create hierarchical structures useful for showing relationships between items, such as category/subcategory relationships, hierarchical navigation, or multi-level outlines. Both ordered and unordered lists can be nested.

### Basic Nested List

```html
<ul>
  <li>Parent item 1
    <ul>
      <li>Child item 1</li>
      <li>Child item 2</li>
    </ul>
  </li>
  <li>Parent item 2</li>
</ul>
```

### Mixed List Types

```html
<ol>
  <li>First category
    <ul>
      <li>Feature A</li>
      <li>Feature B</li>
    </ul>
  </li>
  <li>Second category
    <ul>
      <li>Feature C</li>
      <li>Feature D</li>
    </ul>
  </li>
</ol>
```

### Deep Nesting

```html
<ul>
  <li>Electronics
    <ul>
      <li>Computers
        <ul>
          <li>Laptops</li>
          <li>Desktops</li>
        </ul>
      </li>
      <li>Mobile Devices
        <ul>
          <li>Phones</li>
          <li>Tablets</li>
        </ul>
      </li>
    </ul>
  </li>
</ul>
```

### Real-World Examples

**Navigation Menu:**
```html
<nav>
  <ul>
    <li>Products
      <ul>
        <li>Laptops</li>
        <li>Phones</li>
      </ul>
    </li>
    <li>Services
      <ul>
        <li>Support</li>
        <li>Consulting</li>
      </ul>
    </li>
  </ul>
</nav>
```

### Practice Tips

1. Limit nesting depth (usually 3 levels max)
2. Keep structure logical
3. Use CSS for styling
4. Test accessibility
5. Validate HTML structure
