## Select Dropdowns

**Description**

The `<select>` element creates dropdown menus. Users click to reveal a list of `<option>` elements. Useful for choosing from predefined values like countries, categories, or preferences.

### Basic Select

```html
<select name="color">
  <option value="red">Red</option>
  <option value="green">Green</option>
  <option value="blue">Blue</option>
</select>
```

### With Default Selection

```html
<select name="country">
  <option>Choose a country</option>
  <option value="us" selected>United States</option>
  <option value="uk">United Kingdom</option>
  <option value="ca">Canada</option>
</select>
```

### With Label

```html
<label for="size">Choose Size:</label>
<select id="size" name="size">
  <option value="">-- Select --</option>
  <option value="s">Small</option>
  <option value="m">Medium</option>
  <option value="l">Large</option>
</select>
```

### Option Groups

```html
<select name="animal">
  <optgroup label="Mammals">
    <option value="dog">Dog</option>
    <option value="cat">Cat</option>
  </optgroup>
  <optgroup label="Birds">
    <option value="eagle">Eagle</option>
    <option value="parrot">Parrot</option>
  </optgroup>
</select>
```

### Real-World Examples

**Product Form:**
```html
<form>
  <label for="category">Category:</label>
  <select id="category" name="category" required>
    <option value="">-- Select Category --</option>
    <option value="electronics">Electronics</option>
    <option value="clothing">Clothing</option>
    <option value="books">Books</option>
  </select>
  
  <label for="condition">Condition:</label>
  <select id="condition" name="condition">
    <option value="new">New</option>
    <option value="refurbished">Refurbished</option>
    <option value="used">Used</option>
  </select>
  
  <button type="submit">Submit</button>
</form>
```

### Attributes

- `name` - Form field name
- `id` - Connect to label
- `required` - Must select
- `disabled` - Disable dropdown
- `multiple` - Allow multiple selections

### Practice Tips

1. Use label for accessibility
2. Include default option
3. Group related options
4. Keep lists reasonable length
5. Use value attribute
