## Unordered Lists

**Description**

Unordered lists display items in a bulleted list without any specific order. The `<ul>` element creates the list container, and `<li>` elements define individual list items. They're perfect for features, ingredients, requirements, and any non-sequential information.

### Basic Unordered List

```html
<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>
```

### Nested Lists

```html
<ul>
  <li>Fruits
    <ul>
      <li>Apples</li>
      <li>Oranges</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrots</li>
      <li>Broccoli</li>
    </ul>
  </li>
</ul>
```

### Real-World Examples

**Features List:**
```html
<h2>Our Features</h2>
<ul>
  <li>Easy to use</li>
  <li>Fast performance</li>
  <li>Secure</li>
  <li>24/7 support</li>
</ul>
```

**Requirements:**
```html
<h3>System Requirements</h3>
<ul>
  <li>Windows 10 or later</li>
  <li>4GB RAM minimum</li>
  <li>2GB free disk space</li>
  <li>Internet connection</li>
</ul>
```

### Common Mistakes

❌ **Missing list items:**
```html
<ul>
  First item
  Second item
</ul>
```
✅ **Always use li elements:**
```html
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>
```

### Practice Tips

1. Use for non-sequential lists
2. Keep list items concise
3. Use consistent formatting
4. Nest lists when appropriate
5. Use CSS for custom bullets
