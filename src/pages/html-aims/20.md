## Src & Alt Attributes

**Description**

The `src` attribute specifies the image file location, while the `alt` attribute provides text description for accessibility. Both are essential for proper image implementation. The alt text appears when images fail to load and is crucial for screen readers.

### SRC Attribute

The `src` attribute tells the browser which image file to display:

```html
<!-- Absolute URL -->
<img src="https://example.com/image.jpg">

<!-- Relative path -->
<img src="images/photo.jpg">

<!-- File in same directory -->
<img src="photo.jpg">
```

### ALT Attribute

The `alt` attribute provides descriptive text for the image:

```html
<!-- Descriptive alt text -->
<img src="sunset.jpg" alt="Orange and pink sunset over calm ocean">

<!-- Decorative image (empty alt) -->
<img src="divider.png" alt="">

<!-- Short description -->
<img src="logo.png" alt="Company Logo">
```

### When Image Fails to Load

The `alt` text displays when:
- Image file can't be found
- Network connection is slow
- Browser is text-only
- User disabled image loading

### Accessibility: Why Alt Matters

Screen readers announce alt text to visually impaired users:

```html
<!-- Screen reader: "Golden retriever playing in park" -->
<img src="dog.jpg" alt="Golden retriever playing in park">

<!-- Screen reader: just "image" (not helpful) -->
<img src="dog.jpg" alt="image">

<!-- No alt text (poor accessibility) -->
<img src="dog.jpg">
```

### Alt Text Best Practices

```html
<!-- ✅ Good: Descriptive -->
<img src="team.jpg" alt="Team of five developers working at desks">

<!-- ✅ Good: Specific content -->
<img src="chart.png" alt="Bar chart showing sales increase of 25% in Q3">

<!-- ❌ Bad: Vague -->
<img src="team.jpg" alt="people">

<!-- ❌ Bad: Redundant -->
<img src="team.jpg" alt="image of team">

<!-- ❌ Bad: Too long -->
<img src="team.jpg" alt="This is a picture of our team. There are five people sitting at desks...">
```

### Decorative Images

Decorative images should have empty alt:

```html
<!-- Decorative divider (empty alt) -->
<img src="divider.png" alt="">

<!-- Decorative pattern (empty alt) -->
<img src="pattern.png" alt="">

<!-- But background images in CSS are better for decorative images -->
<div style="background-image: url('pattern.png')"></div>
```

### Real-World Examples

**Product Page:**
```html
<img src="headphones.jpg" alt="Blue wireless headphones with noise cancellation">
```

**Blog Post:**
```html
<img src="article-header.jpg" alt="Person typing on laptop with coffee cup">
```

**Chart:**
```html
<img src="growth-chart.png" alt="Line chart showing company revenue growth from $1M in 2020 to $5M in 2025">
```

**Icon:**
```html
<img src="star.svg" alt="5-star rating">
```

### Common Mistakes

❌ **Missing alt attribute:**
```html
<img src="photo.jpg">
```
✅ **Always include alt:**
```html
<img src="photo.jpg" alt="Description">
```

❌ **Incorrect src path:**
```html
<img src="photo.jpg">  <!-- File not found -->
```
✅ **Correct path:**
```html
<img src="images/photo.jpg">
```

❌ **Redundant alt text:**
```html
<img src="logo.png" alt="logo image">
```
✅ **Concise alt:**
```html
<img src="logo.png" alt="Company Logo">
```

### Image File Formats and Usage

| Format | Best For | Alt Text |
|--------|----------|----------|
| JPG | Photographs | Describe scene |
| PNG | Graphics, logos | Describe content |
| GIF | Animations | Describe animation |
| SVG | Icons, diagrams | Describe function |
| WebP | Modern format | Describe image |

### Testing Alt Text

1. Disable images in browser
2. Use screen reader
3. Read alt text aloud
4. Would text alone convey meaning?

### Practice Tips

1. **Always include alt:** Never skip it
2. **Be descriptive:** Help users understand image
3. **Keep it concise:** Usually 125 characters max
4. **Include context:** What's important about the image?
5. **Test accessibility:** Use screen reader
6. **Validate HTML:** Check for missing attributes
