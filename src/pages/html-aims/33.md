## Text Input Fields

**Description**

Text input fields collect single-line text from users. The `<input type="text">` element creates a basic text box. Use placeholder text to guide users about what to enter.

### Basic Text Input

```html
<input type="text" name="username">
```

### With Placeholder

```html
<input type="text" name="username" placeholder="Enter your username">
```

### With Label

```html
<label for="name">Full Name:</label>
<input type="text" id="name" name="name">
```

### With Attributes

```html
<input 
  type="text" 
  name="username" 
  placeholder="Username"
  required
  maxlength="20"
  value="default">
```

### Real-World Examples

**Sign-up Form:**
```html
<form action="/signup" method="POST">
  <label for="fname">First Name:</label>
  <input type="text" id="fname" name="first_name" required>
  
  <label for="lname">Last Name:</label>
  <input type="text" id="lname" name="last_name" required>
  
  <button type="submit">Sign Up</button>
</form>
```

**Search Box:**
```html
<form action="/search" method="GET">
  <input type="text" name="q" placeholder="Search...">
  <button type="submit">Search</button>
</form>
```

### Useful Attributes

- `placeholder` - Hint text
- `required` - Must fill before submit
- `maxlength` - Max characters allowed
- `minlength` - Min characters required
- `value` - Default value

### Practice Tips

1. Always use labels
2. Set maxlength if appropriate
3. Use placeholder for hints
4. Use required for mandatory fields
5. Style with CSS
