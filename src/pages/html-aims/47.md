## Meta Tags & SEO

**Description**

Meta tags provide metadata about HTML documents in the `<head>` section. They control how pages appear in search engines, social media, and browsers. Essential for SEO and social sharing.

### Essential Meta Tags

```html
<head>
  <!-- Character encoding -->
  <meta charset="UTF-8">
  
  <!-- Viewport (mobile responsiveness) -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <!-- Description for search engines -->
  <meta name="description" content="Brief description of page">
  
  <!-- Keywords -->
  <meta name="keywords" content="html, css, web design">
  
  <!-- Author -->
  <meta name="author" content="John Doe">
</head>
```

### Open Graph Tags (Social Media)

```html
<head>
  <!-- For Facebook, LinkedIn, Twitter -->
  <meta property="og:title" content="Page Title">
  <meta property="og:description" content="Page description">
  <meta property="og:image" content="https://example.com/image.jpg">
  <meta property="og:url" content="https://example.com/page">
  <meta property="og:type" content="website">
</head>
```

### Twitter Card Tags

```html
<head>
  <meta name="twitter:card" content="summary_large_image">
  <meta name="twitter:title" content="Page Title">
  <meta name="twitter:description" content="Page description">
  <meta name="twitter:image" content="https://example.com/image.jpg">
  <meta name="twitter:creator" content="@username">
</head>
```

### Complete Head Example

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <!-- Basic meta tags -->
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Learn web development with our comprehensive guides">
  <meta name="keywords" content="HTML, CSS, JavaScript, web development">
  <meta name="author" content="Web Academy">
  
  <!-- Open Graph for social media -->
  <meta property="og:title" content="Learn Web Development">
  <meta property="og:description" content="Comprehensive web development guides">
  <meta property="og:image" content="https://webacademy.com/og-image.jpg">
  <meta property="og:url" content="https://webacademy.com">
  
  <!-- Twitter Card -->
  <meta name="twitter:card" content="summary_large_image">
  <meta name="twitter:title" content="Learn Web Development">
  <meta name="twitter:description" content="Comprehensive web development guides">
  
  <!-- Other important tags -->
  <meta name="robots" content="index, follow">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="theme-color" content="#3498db">
  
  <!-- Title (not a meta tag, but important) -->
  <title>Learn Web Development | Web Academy</title>
  
  <!-- Links -->
  <link rel="stylesheet" href="styles.css">
  <link rel="canonical" href="https://webacademy.com/learn">
  <link rel="icon" href="favicon.ico">
</head>
<body>
  <!-- Content -->
</body>
</html>
```

### SEO Best Practices

1. **Title** - Unique, descriptive, 50-60 characters
2. **Description** - 150-160 characters, compelling
3. **Keywords** - Relevant but not stuffed
4. **Headings** - Proper h1-h6 hierarchy
5. **Links** - Descriptive link text
6. **Images** - Alt text included

### Real-World Example

```html
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Professional web development services. Build responsive, accessible websites.">
  <meta name="keywords" content="web design, HTML, CSS, JavaScript, responsive design">
  
  <meta property="og:title" content="Expert Web Development Services">
  <meta property="og:description" content="We create beautiful, accessible websites">
  <meta property="og:image" content="https://example.com/preview.jpg">
  
  <title>Web Development Services | Expert Design</title>
  <link rel="canonical" href="https://example.com/services">
</head>
```

### Common Meta Tags

| Tag | Purpose |
|-----|---------|
| charset | Character encoding |
| viewport | Mobile responsiveness |
| description | Search engine snippet |
| keywords | Content keywords |
| og:* | Social media sharing |
| twitter:* | Twitter specific |
| robots | Search engine crawling |
| theme-color | Browser color |

### Practice Tips

1. Write unique descriptions
2. Use relevant keywords
3. Add Open Graph tags
4. Keep title under 60 chars
5. Test with SEO tools
6. Include canonical URLs
7. Optimize for mobile
