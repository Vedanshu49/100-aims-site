## Label Tags

**Description**

Label tags (`<label>`) associate text descriptions with form inputs. They improve accessibility and make form inputs clickable, increasing usability especially on mobile devices.

### Basic Label

```html
<label for="email">Email Address:</label>
<input type="email" id="email" name="email">
```

The `for` attribute connects the label to the input via the input's `id`.

### Label Benefits

1. **Accessibility** - Screen readers read the label
2. **Usability** - Larger clickable area (click label to focus input)
3. **Mobile** - Easier to select on touchscreen
4. **Semantics** - Clear relationship between label and field

### Implicit Label (Nested)

```html
<label>
  Email Address:
  <input type="email" name="email">
</label>
```

### Real-World Examples

**Contact Form:**
```html
<form>
  <label for="name">Full Name:</label>
  <input type="text" id="name" name="name" required>
  
  <label for="phone">Phone Number:</label>
  <input type="tel" id="phone" name="phone">
  
  <label for="message">Message:</label>
  <textarea id="message" name="message"></textarea>
  
  <button type="submit">Send</button>
</form>
```

### Common Mistakes

❌ **Label without for attribute:**
```html
<label>Email:</label>
<input type="email" name="email">
```

✅ **Connect label to input:**
```html
<label for="email">Email:</label>
<input type="email" id="email" name="email">
```

### Practice Tips

1. Always use labels
2. Match for and id attributes
3. Keep labels concise
4. Place labels before inputs
5. Use meaningful text
