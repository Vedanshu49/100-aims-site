## Ordered Lists

**Description**

Ordered lists display items in a numbered sequence. The `<ol>` element creates the list container, and `<li>` elements define individual items. Use ordered lists for steps, rankings, instructions, and any sequential information where order matters.

### Basic Ordered List

```html
<ol>
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</ol>
```

Displays as:
```
1. First step
2. Second step
3. Third step
```

### Numbered Steps

```html
<h2>How to Bake a Cake</h2>
<ol>
  <li>Preheat oven to 350°F</li>
  <li>Mix dry ingredients</li>
  <li>Combine wet ingredients</li>
  <li>Mix until smooth</li>
  <li>Pour into pan</li>
  <li>Bake for 30 minutes</li>
  <li>Cool before serving</li>
</ol>
```

### Top 10 List

```html
<h2>Top 10 Web Browsers</h2>
<ol>
  <li>Chrome</li>
  <li>Firefox</li>
  <li>Safari</li>
  <li>Edge</li>
  <li>Opera</li>
  <li>...and so on</li>
</ol>
```

### Real-World Examples

**Installation Instructions:**
```html
<h3>Setup Instructions</h3>
<ol>
  <li>Download the software</li>
  <li>Run the installer</li>
  <li>Accept the license agreement</li>
  <li>Choose installation directory</li>
  <li>Complete the installation</li>
  <li>Launch the application</li>
</ol>
```

### Common Mistakes

❌ **Using ordered lists for non-sequential:**
```html
<ol>
  <li>Apples</li>
  <li>Oranges</li>
  <li>Bananas</li>
</ol>
```
✅ **Use unordered lists for non-sequential:**
```html
<ul>
  <li>Apples</li>
  <li>Oranges</li>
  <li>Bananas</li>
</ul>
```

### Practice Tips

1. Use for sequential information
2. Use when order matters
3. Use for instructions and steps
4. Keep items concise
5. Test list rendering
