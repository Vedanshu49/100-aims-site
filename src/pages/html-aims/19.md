## Image Tags

**Description**

The `<img>` tag embeds images in web pages. It's a void element (no closing tag) that displays pictures, diagrams, charts, and other visual content. Images are essential for modern websites and require proper attributes for accessibility and performance.

### Basic Image

```html
<img src="photo.jpg" alt="A beautiful sunset">
```

### Image Attributes

```html
<!-- Source (required) -->
<img src="image.jpg">

<!-- Alt text (required for accessibility) -->
<img src="photo.jpg" alt="Description of image">

<!-- Width and height -->
<img src="photo.jpg" alt="Photo" width="300" height="200">

<!-- CSS class for styling -->
<img src="photo.jpg" alt="Photo" class="profile-pic">

<!-- Title (shows on hover) -->
<img src="photo.jpg" alt="Photo" title="My Profile Picture">
```

### Image Paths

```html
<!-- Absolute URL -->
<img src="https://example.com/image.jpg" alt="Image">

<!-- Relative path (same folder) -->
<img src="image.jpg" alt="Image">

<!-- Relative path (subfolder) -->
<img src="images/photo.jpg" alt="Photo">

<!-- Parent directory -->
<img src="../image.jpg" alt="Image">
```

### Image Formats

```html
<!-- JPG (photographs, complex images) -->
<img src="photo.jpg" alt="Photo">

<!-- PNG (graphics, transparent backgrounds) -->
<img src="logo.png" alt="Logo">

<!-- GIF (simple animations) -->
<img src="animation.gif" alt="Animation">

<!-- SVG (scalable vector graphics) -->
<img src="icon.svg" alt="Icon">

<!-- WebP (modern format) -->
<img src="image.webp" alt="Image">
```

### Common Mistakes

❌ **Missing alt text:**
```html
<img src="photo.jpg">
```
✅ **Always include alt:**
```html
<img src="photo.jpg" alt="Description">
```

❌ **Broken image path:**
```html
<img src="photos/image.jpg">  <!-- File doesn't exist -->
```
✅ **Verify file exists:**
```html
<img src="images/photo.jpg" alt="Photo">
```

### Real-World Examples

**Profile Picture:**
```html
<img src="profile.jpg" alt="John Smith's profile picture" width="200" height="200">
```

**Product Image:**
```html
<img src="product.jpg" alt="Blue Wireless Headphones" width="400">
```

### Practice Tips

1. Always use alt text
2. Use appropriate image format
3. Optimize image size
4. Test image paths
5. Use descriptive alt text
