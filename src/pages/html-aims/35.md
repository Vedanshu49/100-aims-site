## Input Types

**Description**

HTML5 provides many input types beyond just text. Each type provides specific validation and keyboard displays on mobile devices. Choose the right type for better user experience.

### Common Input Types

```html
<!-- Text variants -->
<input type="text" placeholder="Basic text">
<input type="email" placeholder="Email address">
<input type="password" placeholder="Secret password">
<input type="search" placeholder="Search term">
<input type="url" placeholder="https://example.com">
<input type="tel" placeholder="Phone number">

<!-- Numbers -->
<input type="number" min="0" max="100">
<input type="range" min="0" max="100">

<!-- Date & Time -->
<input type="date">
<input type="time">
<input type="datetime-local">
<input type="month">
<input type="week">

<!-- Other -->
<input type="color">
<input type="file">
<input type="hidden" value="secret">
```

### Real-World Examples

**Complete Sign-up Form:**
```html
<form action="/signup" method="POST">
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  
  <label for="password">Password:</label>
  <input type="password" id="password" name="password" required>
  
  <label for="phone">Phone:</label>
  <input type="tel" id="phone" name="phone">
  
  <label for="birthdate">Birth Date:</label>
  <input type="date" id="birthdate" name="birthdate">
  
  <label for="age">Age:</label>
  <input type="number" id="age" name="age" min="0" max="150">
  
  <button type="submit">Sign Up</button>
</form>
```

**Event Registration:**
```html
<label for="event-date">Event Date:</label>
<input type="date" id="event-date" name="date" required>

<label for="start-time">Start Time:</label>
<input type="time" id="start-time" name="time">

<label for="attendees">Number of Attendees:</label>
<input type="number" id="attendees" name="count" min="1">
```

### Input Type Benefits

- **Email** - Browser validates email format, mobile shows @ key
- **Tel** - Mobile shows numeric keypad
- **Date** - Browser date picker, easier selection
- **Number** - Numeric only, increment/decrement buttons
- **Range** - Slider for selecting values
- **Color** - Color picker dialog

### Practice Tips

1. Choose appropriate type
2. Mobile keyboards change per type
3. Browsers provide validation
4. Always validate server-side
5. Combine with other attributes
