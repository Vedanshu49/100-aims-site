## Phone Links

**Description**

Phone links use the `tel:` protocol to create clickable telephone numbers. When users click on a phone link from a mobile device, it automatically opens the phone dialer with that number. This is essential for mobile-friendly websites.

### Basic Phone Link

```html
<a href="tel:+1-555-123-4567">Call Us</a>
```

### Phone Format

```html
<!-- With country code -->
<a href="tel:+1-555-123-4567">+1 (555) 123-4567</a>

<!-- Without dashes (simple) -->
<a href="tel:5551234567">555-123-4567</a>

<!-- International format -->
<a href="tel:+44-20-7946-0958">UK Number</a>
```

### With Extension

```html
<a href="tel:+1-555-123-4567x123">Call Extension 123</a>
```

### Real-World Examples

**Contact Card:**
```html
<div class="contact">
  <p>Phone: <a href="tel:+1-555-123-4567">(555) 123-4567</a></p>
  <p>Email: <a href="mailto:contact@example.com">contact@example.com</a></p>
</div>
```

**Call-to-Action:**
```html
<p>
  <a href="tel:+1-800-EXAMPLE">Call 1-800-EXAMPLE</a> for free consultation
</p>
```

### Practice Tips

1. Use full international format with + country code
2. Format for readability in link text
3. Test on mobile devices
4. Use tel: protocol for all phone numbers
