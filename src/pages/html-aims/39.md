## Checkboxes

**Description**

Checkboxes allow users to select multiple options from a group. Unlike radio buttons, any number of checkboxes can be selected. Each checkbox can have a different `name` or share one with array notation.

### Basic Checkboxes

```html
<label>
  <input type="checkbox" name="interests" value="sports">
  Sports
</label>

<label>
  <input type="checkbox" name="interests" value="music">
  Music
</label>

<label>
  <input type="checkbox" name="interests" value="reading">
  Reading
</label>
```

### With Checked State

```html
<label>
  <input type="checkbox" name="terms" value="agree" required>
  I agree to the Terms and Conditions
</label>

<label>
  <input type="checkbox" name="newsletter" value="yes" checked>
  Subscribe to our newsletter
</label>
```

### With Separate Labels

```html
<fieldset>
  <legend>Select your toppings:</legend>
  
  <label for="pepperoni">
    <input type="checkbox" id="pepperoni" name="toppings" value="pepperoni">
    Pepperoni
  </label>
  
  <label for="cheese">
    <input type="checkbox" id="cheese" name="toppings" value="cheese" checked>
    Extra Cheese
  </label>
  
  <label for="mushrooms">
    <input type="checkbox" id="mushrooms" name="toppings" value="mushrooms">
    Mushrooms
  </label>
</fieldset>
```

### Real-World Examples

**Preferences Form:**
```html
<form action="/preferences" method="POST">
  <fieldset>
    <legend>Communication Preferences:</legend>
    
    <label>
      <input type="checkbox" name="email" value="yes">
      Send me emails
    </label>
    
    <label>
      <input type="checkbox" name="sms" value="yes">
      Send me SMS messages
    </label>
    
    <label>
      <input type="checkbox" name="phone" value="yes">
      Call me
    </label>
  </fieldset>
  
  <button type="submit">Save Preferences</button>
</form>
```

### Key Differences

| Aspect | Radio | Checkbox |
|--------|-------|----------|
| Selection | One only | Multiple |
| Same group | Same name | Same name |
| Use case | Exclusive choice | Multiple choices |
| Default | checked | checked |

### Practice Tips

1. Use labels for each checkbox
2. Use same name for groups
3. Provide clear options
4. Use value attribute
5. Test form submission
