## Email Links

**Description**

Email links allow users to send emails directly from your website. Using the `mailto:` protocol in an href attribute, you can create links that open the user's default email client. This is useful for contact forms and communication links.

### Key Concepts

- **mailto:** Protocol for email links
- **Email address:** Required in href
- **Subject line:** Optional parameter
- **Email client:** Opens user's default email application
- **Alternative:** Forms are more flexible for modern websites

### Basic Email Link

```html
<!-- Simple email link -->
<a href="mailto:contact@example.com">Email Us</a>

<!-- Different email address -->
<a href="mailto:info@company.com">Contact Info</a>
```

### Email with Subject

```html
<!-- Email with subject line -->
<a href="mailto:contact@example.com?subject=Website%20Inquiry">
  Email Us
</a>

<!-- Spaces in subject (use %20) -->
<a href="mailto:support@example.com?subject=Need%20Technical%20Support">
  Get Technical Support
</a>
```

### Email with Subject and Body

```html
<!-- Email with subject and message body -->
<a href="mailto:apply@company.com?subject=Job%20Application&body=Hello,%20I%20am%20interested%20in%20applying%20for%20a%20position.">
  Apply Now
</a>
```

### Multiple Recipients

```html
<!-- Multiple email addresses (separated by comma) -->
<a href="mailto:to@example.com,cc@example.com?subject=Meeting">
  Email Team
</a>
```

### Real-World Examples

**Contact Link:**
```html
<footer>
  <a href="mailto:contact@example.com">
    contact@example.com
  </a>
</footer>
```

**Support Request:**
```html
<a href="mailto:support@company.com?subject=Help%20with%20Account&body=I%20need%20help%20with%20my%20account.">
  Request Support
</a>
```

**Feedback Form:**
```html
<a href="mailto:feedback@company.com?subject=Website%20Feedback">
  Send Feedback
</a>
```

### Common Mistakes

❌ **Missing mailto: protocol:**
```html
<a href="contact@example.com">Email</a>
```
✅ **Include mailto: protocol:**
```html
<a href="mailto:contact@example.com">Email</a>
```

### Character Encoding

Special characters need encoding:

```html
<!-- Ampersand in body -->
<a href="mailto:email@example.com?subject=Query&body=What%20is%20this?">
  Email
</a>

<!-- Space encoded as %20 -->
<a href="mailto:email@example.com?subject=Important%20Message">
  Email
</a>
```

### Practice Tips

1. **Always use mailto: protocol**
2. **Encode spaces as %20**
3. **Consider form alternatives**
4. **Test email links**
5. **Provide clear contact options**
