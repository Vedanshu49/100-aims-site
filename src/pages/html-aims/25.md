## Definition Lists

**Description**

Definition lists (`<dl>`) associate terms with their definitions. They use `<dt>` for terms and `<dd>` for definitions. Perfect for glossaries, dictionary entries, and any term-definition pairs.

### Basic Definition List

```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
  
  <dt>JavaScript</dt>
  <dd>A programming language for web browsers</dd>
</dl>
```

### Real-World Examples

**Glossary:**
```html
<h2>Web Development Glossary</h2>
<dl>
  <dt>Semantics</dt>
  <dd>Using elements that carry meaning about the content they contain</dd>
  
  <dt>Responsive Design</dt>
  <dd>Designing websites to work on all device sizes</dd>
  
  <dt>Accessibility</dt>
  <dd>Making websites usable by everyone, including people with disabilities</dd>
</dl>
```

**FAQ:**
```html
<dl>
  <dt>What is HTML?</dt>
  <dd>HTML is the standard markup language for creating web pages</dd>
  
  <dt>Why learn HTML?</dt>
  <dd>HTML is the foundation of web development and essential for all web professionals</dd>
</dl>
```

### Practice Tips

1. Use for term-definition pairs
2. Use for glossaries
3. Group related terms
4. Keep definitions clear
5. Use semantic markup
