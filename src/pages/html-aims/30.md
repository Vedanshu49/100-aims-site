## Table Cell Merging

**Description**

Merge table cells horizontally with `colspan` or vertically with `rowspan`. This creates complex table layouts for grouped data or headers.

### Colspan - Merge Columns

```html
<table>
  <tr>
    <th colspan="2">Personal Info</th>
    <th colspan="2">Contact</th>
  </tr>
  <tr>
    <th>First</th>
    <th>Last</th>
    <th>Email</th>
    <th>Phone</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>john@example.com</td>
    <td>555-1234</td>
  </tr>
</table>
```

### Rowspan - Merge Rows

```html
<table>
  <tr>
    <th>Day</th>
    <th>Time</th>
    <th>Activity</th>
  </tr>
  <tr>
    <td rowspan="2">Monday</td>
    <td>9 AM</td>
    <td>Breakfast</td>
  </tr>
  <tr>
    <td>12 PM</td>
    <td>Lunch</td>
  </tr>
  <tr>
    <td>Tuesday</td>
    <td>10 AM</td>
    <td>Meeting</td>
  </tr>
</table>
```

### Combined Merging

```html
<table>
  <tr>
    <th colspan="2" rowspan="2">2x2 Cell</th>
    <th>Col 3</th>
  </tr>
  <tr>
    <th>Another</th>
  </tr>
  <tr>
    <td>Row 2 Cell 1</td>
    <td>Row 2 Cell 2</td>
    <td>Row 2 Cell 3</td>
  </tr>
</table>
```

### Real-World Example

**Conference Schedule:**
```html
<table>
  <tr>
    <th>Time</th>
    <th colspan="2">Track A & B</th>
  </tr>
  <tr>
    <td>9:00 AM</td>
    <td>Opening Keynote</td>
    <td>Welcome</td>
  </tr>
  <tr>
    <td>10:00 AM</td>
    <td>Web Dev Talk</td>
    <td>Design Workshop</td>
  </tr>
</table>
```

### Practice Tips

1. Use colspan/rowspan carefully
2. Keep tables readable
3. Avoid complex merging
4. Test accessibility
5. Consider alternatives (list of tables)
