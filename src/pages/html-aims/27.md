## Basic Table Structure

**Description**

Tables consist of rows (`<tr>`) and cells. `<td>` elements are data cells, while `<th>` elements are header cells. Every table needs this basic structure.

### Table Elements

```html
<table>
  <tr>        <!-- Table Row -->
    <td>Cell 1</td>  <!-- Table Data -->
    <td>Cell 2</td>
  </tr>
  <tr>
    <td>Cell 3</td>
    <td>Cell 4</td>
  </tr>
</table>
```

### With Headers

```html
<table>
  <tr>
    <th>Name</th>      <!-- Table Header -->
    <th>Email</th>
    <th>Phone</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>alice@example.com</td>
    <td>555-1234</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>bob@example.com</td>
    <td>555-5678</td>
  </tr>
</table>
```

### Real-World Example

```html
<h2>Student Grades</h2>
<table>
  <tr>
    <th>Student</th>
    <th>Math</th>
    <th>Science</th>
    <th>English</th>
  </tr>
  <tr>
    <td>Sarah</td>
    <td>92</td>
    <td>88</td>
    <td>95</td>
  </tr>
  <tr>
    <td>Michael</td>
    <td>85</td>
    <td>90</td>
    <td>87</td>
  </tr>
</table>
```

### Common Mistakes

❌ **Using only td without headers:**
```html
<table>
  <tr>
    <td>Name</td>
    <td>Email</td>
  </tr>
</table>
```

✅ **Use th for headers:**
```html
<table>
  <tr>
    <th>Name</th>
    <th>Email</th>
  </tr>
</table>
```

### Practice Tips

1. Use th for headers
2. Use td for data
3. Keep tables simple
4. One row per tr element
5. One cell per td/th element
