## Image Dimensions

**Description**

You can specify image dimensions using the `width` and `height` attributes. Setting explicit dimensions improves page performance by reserving space for images before they load, preventing layout shift. You can also use CSS for more flexible sizing.

### Width and Height Attributes

```html
<!-- Explicit dimensions in pixels -->
<img src="photo.jpg" alt="Photo" width="300" height="200">

<!-- Only width (height auto-adjusts) -->
<img src="photo.jpg" alt="Photo" width="400">

<!-- Only height (width auto-adjusts) -->
<img src="photo.jpg" alt="Photo" height="300">

<!-- No dimensions (uses image's natural size) -->
<img src="photo.jpg" alt="Photo">
```

### Responsive Images with CSS

```html
<style>
  /* Image takes full width of container -->
  img {
    max-width: 100%;
    height: auto;
  }
</style>

<img src="photo.jpg" alt="Photo">
```

### Common Image Sizes

```html
<!-- Thumbnail -->
<img src="photo.jpg" alt="Thumbnail" width="150" height="150">

<!-- Post feature image -->
<img src="header.jpg" alt="Post header" width="800" height="400">

<!-- Profile picture -->
<img src="profile.jpg" alt="Profile" width="200" height="200">

<!-- Full-width image -->
<img src="banner.jpg" alt="Banner" width="1200" height="300">
```

### Performance: Layout Shift Prevention

```html
<!-- Good: Explicit dimensions prevent layout shift -->
<img src="photo.jpg" alt="Photo" width="400" height="300">

<!-- Issue: No dimensions, page may shift when image loads -->
<img src="photo.jpg" alt="Photo">
```

### Practice Tips

1. Set width and height to prevent layout shift
2. Use responsive sizing with CSS
3. Test on different screen sizes
4. Optimize image file sizes
5. Use appropriate dimensions for use case
