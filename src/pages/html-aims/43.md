## Required & Disabled Attributes

**Description**

The `required` attribute makes form fields mandatory. The `disabled` attribute prevents users from interacting with a field. These are common HTML5 attributes for form control.

### Required Attribute

```html
<!-- Text must be filled -->
<input type="text" name="username" required>

<!-- Email must be provided -->
<input type="email" name="email" required>

<!-- Checkbox must be checked -->
<input type="checkbox" name="agree" required>
```

### Disabled Attribute

```html
<!-- User cannot interact -->
<input type="text" name="readonly" disabled>

<!-- Disabled button -->
<button type="submit" disabled>Processing...</button>

<!-- Disabled select -->
<select name="country" disabled>
  <option>Choose...</option>
</select>
```

### Common Use Cases

**Required:**
```html
<form action="/submit" method="POST">
  <label for="email">Email Address:</label>
  <input type="email" id="email" name="email" required>
  
  <label for="password">Password:</label>
  <input type="password" id="password" name="password" required>
  
  <button type="submit">Login</button>
</form>
```

**Disabled:**
```html
<!-- Conditional field (disabled until option selected) -->
<label for="shipping">Shipping Method:</label>
<select id="shipping" name="shipping">
  <option>Standard</option>
  <option>Express</option>
</select>

<label for="insurance">Add Insurance:</label>
<input type="checkbox" id="insurance" name="insurance" disabled>
```

### Real-World Examples

**Sign-up Form:**
```html
<form>
  <input type="text" name="first_name" required placeholder="First Name">
  <input type="text" name="last_name" required placeholder="Last Name">
  <input type="email" name="email" required placeholder="Email">
  <input type="password" name="password" required placeholder="Password">
  
  <label>
    <input type="checkbox" name="terms" required>
    I agree to Terms (required)
  </label>
  
  <button type="submit">Sign Up</button>
</form>
```

**Processing State:**
```html
<button type="submit" id="btn">Submit</button>

<script>
  document.getElementById('btn').addEventListener('click', () => {
    document.getElementById('btn').disabled = true;
    document.getElementById('btn').textContent = 'Processing...';
  });
</script>
```

### Key Points

- **required** - Prevents form submission if empty
- **disabled** - Element appears grayed out, not submitted
- Both improve UX and accessibility
- Browser provides validation feedback

### Practice Tips

1. Use required for mandatory fields
2. Use disabled for conditional fields
3. Provide visual feedback
4. Always validate server-side
5. Test accessibility
