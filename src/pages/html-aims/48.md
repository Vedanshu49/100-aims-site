## Favicon & Site Icons

**Description**

Favicons are small icons displayed in browser tabs and bookmarks. Modern websites use multiple icon sizes for different devices and platforms. Links go in the `<head>` section using `<link>` tags.

### Basic Favicon

```html
<head>
  <!-- Traditional favicon -->
  <link rel="icon" href="favicon.ico">
  
  <!-- PNG format (modern) -->
  <link rel="icon" type="image/png" href="favicon.png">
  
  <!-- SVG format (scalable) -->
  <link rel="icon" type="image/svg+xml" href="favicon.svg">
</head>
```

### Multiple Icon Sizes

```html
<head>
  <!-- Standard favicon -->
  <link rel="icon" sizes="16x16" href="favicon-16.png">
  <link rel="icon" sizes="32x32" href="favicon-32.png">
  
  <!-- Apple touch icon (iOS) -->
  <link rel="apple-touch-icon" href="apple-touch-icon.png">
  
  <!-- Android -->
  <link rel="icon" sizes="192x192" href="android-192.png">
  <link rel="icon" sizes="512x512" href="android-512.png">
</head>
```

### Complete Icon Setup

```html
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <!-- Browser tab icon -->
  <link rel="icon" type="image/svg+xml" href="favicon.svg">
  <link rel="icon" type="image/png" sizes="32x32" href="favicon-32.png">
  <link rel="icon" type="image/png" sizes="16x16" href="favicon-16.png">
  
  <!-- Apple devices (iPhone, iPad) -->
  <link rel="apple-touch-icon" sizes="180x180" href="apple-touch-icon.png">
  
  <!-- Android and other devices -->
  <link rel="icon" sizes="192x192" href="icon-192.png">
  <link rel="icon" sizes="512x512" href="icon-512.png">
  
  <!-- PWA manifest -->
  <link rel="manifest" href="site.webmanifest">
  
  <!-- Theme color for browser chrome -->
  <meta name="theme-color" content="#3498db">
  
  <title>My Website</title>
</head>
```

### Real-World Examples

**Minimal Setup:**
```html
<head>
  <link rel="icon" href="favicon.svg">
  <link rel="apple-touch-icon" href="apple-icon.png">
</head>
```

**Complete Setup:**
```html
<head>
  <!-- SVG favicon (preferred) -->
  <link rel="icon" type="image/svg+xml" href="/favicon.svg">
  
  <!-- Fallback PNG -->
  <link rel="icon" type="image/png" href="/favicon-32.png" sizes="32x32">
  
  <!-- Apple touch icon -->
  <link rel="apple-touch-icon" href="/apple-icon-180.png" sizes="180x180">
  
  <!-- Android -->
  <link rel="icon" sizes="192x192" href="/android-chrome-192.png">
  <link rel="icon" sizes="512x512" href="/android-chrome-512.png">
  
  <!-- Theme color -->
  <meta name="theme-color" content="#ffffff">
  <meta name="msapplication-TileColor" content="#ffffff">
  
  <title>My Website</title>
</head>
```

### Icon Formats

| Format | Best For | Size |
|--------|----------|------|
| .ico | Fallback | 16-64px |
| .png | Modern browsers | 32-512px |
| .svg | Scalable, modern | Any size |
| .webp | Progressive | 32-512px |

### Practice Tips

1. Use SVG when possible (scalable)
2. Provide multiple sizes
3. Include Apple touch icon
4. Set theme color
5. Use consistent branding
6. Test on different devices
